# Target design system — Vibe 课堂作业平台

Use a light, calm, classroom-friendly Chinese web application visual language. This replaces the current dark developer-console look.

- Font: system sans-serif, prioritize `PingFang SC`, `Microsoft YaHei`, `Noto Sans SC`, then `Segoe UI`.
- Background: warm off-white `#F8FAFC`; surfaces `#FFFFFF`; muted surface `#F1F5F9`.
- Ink: `#172033`; secondary text `#64748B`; border `#E2E8F0`.
- Primary: academic blue `#2563EB`; deep blue `#1D4ED8`; success `#15803D`; warning `#B45309`; danger `#B91C1C`.
- Radius: 12px cards, 8px controls, soft low-elevation shadows only.
- Layout: responsive 1200px desktop workspace; high information density with generous whitespace; strong course and assignment hierarchy.
- Teacher pages: a clean blue-and-white top bar, cards, readable tables, explicit state labels and accessible contrast.
- Classroom presentation: bright, full-screen, large type, and student work first. Show group name, members, screenshots, project/code highlights, and process metrics. Do not show grades, evaluation text, flags, or feedback.
- Private teacher review: evaluation and feedback live in a separate review surface, never in the projector view.
